# react-hooks-testing-library-playground

A playground for the amazeballs [react-hooks-testing-library](testing-library/react-hooks-testing-library) ([Documentation](https://react-hooks-testing-library.com)).

## Usage


### Install
```sh
# if you use yarn
$ yarn install

# if you use npm:
$ npm install
```

### Run tests
```sh
# if you use yarn
$ yarn test

# if you use npm:
$ npm test
```
